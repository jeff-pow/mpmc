These are sample inputs for fitting noble gases in MPMC.

Use, for example,

/path/to/mpmc helium_fitting.input

to run.

All 3 files are required to run PES:
	- *.input (MPMC input)
	- *.pqr (initial parameters/coordinates)
	- *.dat (PES input data)
